14.08.2010Neue Mau Dau Mau Version 1.19
Im Downloadbereich befindet sich nun die neue Version 1.19.105 Build 3808 von Mau Dau Mau.

Die neue Version behebt einige Fehler, die aber noch beim Spielen mit Spielern, die eine alte Version benutzen (abwärtskompatibel), auftreten können (unter bestimmten seltenen Umständen).

Zusätzlich hat die neue Version zwei weitere Sortieroptionen, um Aktionskarten gesondert anzuordnen.

Viel Spaß.
30.07.2010Medium Size Thumbnails
Bisher war es so, das die Bilder in der Übersicht kleine Thumbnails (max. 169x169px) waren, und nachdem man auf ein Bild klickte, wurde das Bild dynamisch in voller Größe (unbegrenzt) runtergeladen und auf die Bildschirmgröße (JavaScript) runterskaliert.

Dadurch hat es vorallem bei größeren Bildern lange gedauert sie anzuzeigen.

Jetzt habe ich eine Zwischenstufe eingebaut, so dass nicht mehr die riesigen Bilder dynamisch geladen und skaliert werden, sondern etwas größere Thumbnails (max. 719x719px). Bei einem Klick auf diese wird das Bild in Originalgröße geöffnet.

Außerdem sind die USA-Bilder nun in besserer (Original-)Qualität vorhanden (hatte sie "damals" runterskaliert).
